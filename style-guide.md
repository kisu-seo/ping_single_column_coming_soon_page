### Text Presets

- **Preset 1**: 48px / 125% Line Height / 0px Letter Spacing
  - Bold (700), Light (300)
- **Preset 2**: 20px / 125% Line Height / 0px Letter Spacing
  - Bold (700), Light (300)
- **Preset 3**: 16px / 125% Line Height / 0px Letter Spacing
  - SemiBold (600), Light (300)
- **Preset 4**: 12px / 125% Line Height / 0px Letter Spacing
  - SemiBold (600), Light (300), Italic
- **Preset 5**: 10px / 125% Line Height / 0px Letter Spacing
  - Italic
